# DROPSCHEMA

Drop a non-public schema and all AI stores inside it. Ahnlich AI also drops the backing DB schema first. `public` cannot be dropped.

## Syntax

```bash
DROPSCHEMA <schema>
```

## Example

```bash
DROPSCHEMA media
```

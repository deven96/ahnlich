# GETKEY

Retrieve items by their original input key — available when the store preserves originals (`STOREORIGINAL`).

## Syntax

```bash
GETKEY <key> IN <store>
GETKEY <key> IN <store> SCHEMA media
```

## Example

```bash
GETKEY doc1 IN article_store
```

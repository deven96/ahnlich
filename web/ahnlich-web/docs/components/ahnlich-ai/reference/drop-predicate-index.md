# DROPPREDICATEINDEX

Remove a previously created metadata index.

## Syntax

```bash
DROPPREDICATEINDEX <field> IN <store>
DROPPREDICATEINDEX <field> IN <store> SCHEMA media
```

## Example

```bash
DROPPREDICATEINDEX category IN article_store
```

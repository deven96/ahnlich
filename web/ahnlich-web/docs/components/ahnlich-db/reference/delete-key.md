# DELETE KEY

Delete a vector by its exact key.

## Syntax

```bash
DELETE KEY <key> IN <store_name>
DELETE KEY <key> IN <store_name> SCHEMA analytics
```

## Example

```bash
DELETE KEY doc1 IN my_store
```

# GETSTORE

Get the same details as LISTSTORES, but for a single store. Errors if it doesn't exist.

## Syntax

```bash
GETSTORE <store_name>
GETSTORE <store_name> SCHEMA analytics
```

## Example

```bash
GETSTORE my_store
```
